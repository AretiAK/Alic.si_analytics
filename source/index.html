Alice powers the social finance ecosystem

Alice is a blockchain-based infrastructure that makes payments conditional to verified impact. It allows anyone to create and monetise transparent social funding applications. Examples of applications built on Alice include:

  • Scalable Project Finance

  • Automated impact tracking

  • Transparent donations

  • Tradeable impact investments

  • Beneficiary accountability tools
